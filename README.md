# FUTURE_CS_03
# API Security Risk Analysis

## Objective

This project analyzes a public API to identify common API security risks and provide remediation recommendations.

## API Tested

JSONPlaceholder API

https://jsonplaceholder.typicode.com

## Tools Used

* Postman
* Browser Developer Tools
* GitHub

## Methodology

1. Reviewed API documentation
2. Tested API endpoints
3. Inspected responses and headers
4. Analyzed authentication
5. Reviewed authorization controls
6. Documented findings

## Endpoints Tested

* /posts
* /users
* /comments

## Key Findings

* Unauthenticated Access
* Excessive Data Exposure
* Broken Object Level Authorization (BOLA)
* Missing Security Headers
* Missing Rate Limiting
